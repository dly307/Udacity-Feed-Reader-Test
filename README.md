# Project Overview

This particular project, we are given a web-based application that reads RSS feeds. This project already includes [Jasmine](http://jasmine.github.io/) and there are many test suites. 


## Instructions

In order to open the code, open up the index.html file in a browser and it should display the Udacity Blog.  At the bottom of the page should be Jasmine with seven test suites.  

## Test Suites for this Project

Test #1:  Ensures that all of the feeds shown on the html have URLs and are defined and working properly.

Test #2:  Guarantees that the hamburger menu element of the website toggles on (show) and off (hide) as it should when clicked and clicked again, respectively.  

Test #3:  Certifies that the a new feed is loaded for the initial entries that are clicked.
